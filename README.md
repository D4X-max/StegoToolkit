# StegoToolkit: Advanced Steganography Suite

**StegoToolkit** is a high-performance security application designed for hiding and detecting encrypted data within digital media.

##  Key Features

* **Image Steganography:** Hide encrypted text inside PNG/BMP files.
* **AES-256 Security:** Payloads are encrypted using AES-256-CBC with PKCS7 padding.
* **Forensic Anomaly Scanner:** Generates **Entropy Spatial Heatmaps** to detect hidden data.

##  How to Run

### 1. Installation
```bash
pip install flask numpy Pillow cryptography
```
### 2.Launching the project
```bash
python app.py
```
