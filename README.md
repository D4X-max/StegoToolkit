🛡️ StegoToolkit: Advanced Steganography SuiteStegoToolkit is a high-performance security application designed for hiding and detecting encrypted data within digital media. Utilizing AES-256-CBC encryption and NumPy-optimized vectorized LSB (Least Significant Bit) injection, it offers a robust platform for secure communication and forensic image analysis.✨ Key FeaturesImage Steganography: Hide encrypted text inside PNG/BMP files using bit-plane manipulation.AES-256 Security: All payloads are encrypted using industry-standard AES-256-CBC with PKCS7 padding and PBKDF2 key derivation.Forensic Anomaly Scanner: A heuristic model that generates Entropy Spatial Heatmaps to detect hidden data in suspicious images.PDF Metadata Vault: Inject secret strings into PDF document metadata fields.Batch Processing: Securely encode multiple images simultaneously and download them as a ZIP archive.Real-time Capacity Analysis: Dynamically calculates how many bytes an image can safely store without visual degradation.🚀 How to Run1. By flattening the image matrix and applying bitmasks, we ensure that the execution time remains under 2 seconds even for high-resolution 4K assets.Forensic Detection LogicThe Anomaly Scanner uses a sliding window entropy check. It analyzes $16 \times 16$ pixel blocks to calculate the statistical density of the LSB plane. Because AES-encrypted data has a near-perfect 0.5 bit-density, the toolkit can differentiate between natural image noise and artificial ciphertext.📂 Project Structureapp.py: Flask server and API routing.stego/lsb.py: Core steganography and forensic logic.crypto/aes.py: Encryption and decryption modules.templates/: HTML5/TailwindCSS frontend.uploads/: Temporary storage for analysis (auto-cleaned).outputs/: Generated stego-images and heatmaps.
